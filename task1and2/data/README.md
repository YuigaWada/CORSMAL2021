# Dataset
Download & put the dataset here.  
**e.g.**  
```sh
data/ $ unzip ~/Downloads/ccm_train_view1_rgb.zip ./
```

```
data/

  train/  # this directory is compressed in every zip file on the official dataset page
    audio/
```
